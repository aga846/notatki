# Co robi if  
If sprawdza, czy dane wyrażenie jest prawdziwe.  
  
```
name = "Damian"
if name            <-- to jest True

l = []
if l               <-- to jest False

if not l           <-- to jest True
```
  
True jest wynikiem sprawdzenia, jeśli string, lista, itd., nie są puste.  
Domyślnie False są:  
- None,  
- 0,  
- puste stringi, listy, obiekty.
